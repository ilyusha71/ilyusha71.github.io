2014-06-05 『入侵模式』第一階段測試開始

「入侵模式」已經開始封閉測試。

許多飛行員已經聽說過這種模式，我們之前在論壇提到過它。

在第一階段期間下面的飛行員將有權限測試「入侵模式」：

  • 受到特別邀請的玩家
  • 獲得「Prometheus Squad」成就的玩家
  • 獲得「Universe Conqueror」成就的玩家

要進入新的模式，請按「Launch」，然後選擇「Attacked Sectors」標籤。

「Universe Conqueror」成就需要收集1000點的成就點數。

「Strikeforce Prometheus」成就需要購買以下的DLC：

  • 6000 + 900 (bonus) Galactic Standards
  • 12500 + 2500 (bonus) Galactic Standards
  • 35000 + 7500 (bonus) Galactic Standards
  • Pirate Pack — Jolly Roger
  • Pirate Pack — Dead man's chest
  • Mercenary Pack — Galaxy Explorer
  • Mercenary Pack — Elite Pilot

以下這些是steam平台可購買到的DLC

  • Galaxy Explorer Pack
  • Elite Pilot Pack
  • Pirate Pack - Jolly Roger
  • Pirate Pack - Dead Man's Chest

更多詳細信息可以在我們的論壇上找到。

—星際爭端 小組—

2014-06-05 四階新機『切割者』新登場！

傭兵，從今天開始，你可能在戰場上會面臨新的敵人 - T4浩劫特勤機準備升空！

它具有一系列獨特的特點：

  • 帶來的雙重收益
  • 不需要維修
  • 有兩個額外的貨物插槽
  • 累積20％的自由協同作用
  • 信標捕獲速度提高20％
  • 爆擊機率提高30％
  • 主要武器造成的傷害提高10％
  • 特勤機裝備效果提高20％

隱形特勤艦船是設計給打帶跑來攻擊敵機的玩家。

一個特殊的等離子網絡能使敵人在沒有任何直接接觸時造成傷害，而船舶的制導系統則給予更多的爆擊。

特勤機可以裝備暫時增加傷害的主要武器，以及防止鎖定你船艦的敵人。

與「切割者」削減你的敵人吧！

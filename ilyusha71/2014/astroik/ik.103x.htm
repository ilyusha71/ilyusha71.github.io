　　曆法，是一部人類遺留至今仍在使用的工具，最早的起源可以追溯至遠古時期，當時人們藉由日月的出現來記錄時間，因整個天空最明顯的物體即是太陽，因此日的概念是最早誕生的，人類進入文明時代後發明了日晷，得以演變成今日時、分與秒的計時方式，隨之發展成熟的是月與年的概念，歷史上絕大多數的曆法都離不開這兩者，除了少數的曆法，如Cholqij ( 一種瑪雅曆法 ) 。

　　在漫長的歲月裡，世界各地不同的民族擁有不一樣的文化色彩，以致於演變成各種不同類型的曆法，如我國至今仍在使用中國傳統的曆法，也是經歷過數個朝代的改革而流傳下來的，至今世界上所使用的曆法主要可分為三大類，陰曆、陽曆與陰陽合曆。

一、 日的概念 ( Day )

　　自古以來，人類就已發現日月運行的規律，隨之而然地，日的概念最初形成，稱作一天，在一天中會有明顯地晝夜之分，即是判定一天時間的依據，今日我們都知道這是由於地球自轉造成的現象，天文學上，太陽兩次通過上中天的時間，稱作一個真太陽日，但實際上地球的自轉週期並非固定的，時快時慢，必須每天藉由精密儀器的測量才可得知，為了能近於人們在生活上使用，當時的一日即是所謂平太陽的概念，將每個真太陽日的平均長度，定為一個平太陽日，今日的陽曆即是以平太陽日作為一日長度的單位，相當於24小時。

　　但是地球真正的自轉週期並非為一個太陽日，這是由於地球自轉的同時仍在公轉 ( 見下方圖解 ) ，亦即太陽在天球上的位置並非是固定的，換言之，天球上的恆星其距離都比太陽遙遠得多，其每天的自行運動很小 ( 可視為固定不動 ) ，經由測量某顆恆星兩次通過上中天所經歷的時間，可得知地球的自轉週期實際上只有23小時56分鐘04秒，這個週期被定義為一個恆星日， 嚴謹來說恆星日是以天球上的春分點做為參考點，而非恆星，但另一方面春分點也因歲差的關係不斷地退行，雖然每天移動的幅度非常小，但這說明了地球自轉的週期與一個恆星日還是有所差別。

　　假設在9月22日的午夜十二時，昴宿星團 ( 本例子以昴宿星團作為天球上的一個參考星 ) 位在上中天的位置 ( 粉紅色箭頭所指 ) 。隨著地球的自轉與公轉，昴宿星團又再度回到上中天時 ( 綠色的箭頭所指 ) ，這之間所經歷的時間即為一個恆星日。

　　而在地球的另一側是9月22日的中午十二點，此時太陽位在上中天的位置 ( 紅色箭頭所指 ) ，在一個恆星日之後，太陽會移動到中天子午線 ( 黃色箭頭所指 ) 的東邊，這是由於地球自轉的同時也在公轉，每天地球公轉大約1度 ( 紅色與淺藍色箭頭之夾角，太陽日始-太陽日終 ) ，相對於太陽的黃經也會增加1度 ( 順行 ) ，因此地球自轉就必須再多轉1度 ( 黃色與淺藍色箭頭之夾角，恆星日終-太陽日終 ) ，太陽才會回到中天子午線上 ( 淺藍色箭頭所指 ) ，在這整段的週期即是一個太陽日。

　　綜合太陽日與恆星日的原理，我們即可得知為何每天星星會提早四分鐘升起，下方的動畫即為一個實際例子。

二、 月的概念 ( Month )

　　月球是距離地球最近的天體，也是地球唯一已知的天然衛星，由於易於觀測，人類文明在很早即已發現月相具有規律的變化，月相的週期大致上已被測量出，即是陰曆的基礎，朔望月。

　　朔望月被定義為兩次合朔的週期，合朔是指月相從朔再變成朔的時候，這之間的時間實際上受到地球重力場產生的攝動而並不穩定，但是影響週期幅度不大，因此朔望月的週期平均大約為29.5天。

　　即使曆法上採用的是朔望月，但朔望月並不是月球實際公轉地球的週期，這是由於月球公轉地球的同時，地球也同時在公轉，導致一個月會相差將近30度，為了確認這個週期，將背景星空的某一天體作為參考點，從地球上觀測月球在星空的視運動，兩次回到相同位置時即為實際的公轉週期。這個原理，可以假想成我們現在位於遙遠的某一顆星星上，直接觀測月球的公轉週期。因此這個週期稱作恆星月，其長度約為27.3天。

　　 望是月球相對於太陽相位差180度的時候，假設在某一天的午夜十二點，正是望的時候 ( 粉紅色箭頭所指 ) ，同時相對於昴宿星團位在其下方2度的位置 ( 本例子以昴宿星團作為天球上的一個參考星 ) ，隨著地球與月球各自的公轉，月球再度回到 昴宿星團附近時 ( 綠色箭頭所指 ) ，這之間所經歷的時間即為一個恆星月，但是在月球公轉的同時地球也在公轉 ，這段時間內地球公轉了將近30度左右，相對於太陽黃經增加了30度 ( 順行 ) ，使得月球與太陽的相位差只有150度 ( 黃色虛線箭頭所指 ) ，因此月球必須在恆星月之後多運行30度，才能與太陽相距180度而變成望的月相 ( 淺藍色的箭頭所指 ) ， 而在這兩次望之間所經歷的時間即為一個朔望月。

三、 陰曆 ( Lunar calendar )

　　陰曆是人類歷史上最早的曆法，是根據月亮的盈虧變化來紀月，所以陰曆完整來說應稱作太陰曆，在陰曆中，一個月的長度相當於一個朔望月，朔為一月之始，望為一月之中，朔望月即是經歷兩次朔之間的時間，而實際上朔望月的平均週期為29.530589天，並不為整數日，以致於陰曆月的長度有29天與30天，同時朔望月的週期也會有些許變化，加上月球的運行並非等速，因此導致陰曆十五不一定是滿月，而有可能會在陰曆十六日、十七日。

　　陰曆的閏年，如同陽曆的閏年，因每個陰曆年年的實際長度不為一個整數天，而需置閏，但是不要與陰陽曆的閏月混為一談，一個陰曆年的實際長度約為354.368268，其中的小數部分接近11/30，所以在閏年上，每三十年會置十一個閏年，亦即陰曆閏年會有355天，平年則為354天。

　　閏月是只有陰陽曆才有的，如農曆，純太陽曆與太陰曆只有閏年，而陰陽曆中包含閏月的那一年也稱作閏年。

五、 陰陽合曆 ( Lunisolar calendar )

　　陰陽曆為陰陽合曆的簡稱，為了同時考慮太陽與太陰的運行，並使陰曆年能接近陽曆年的長度，促使了陰陽曆的誕生，為了達到陰陽合曆的目的，在陰曆中引進了閏月的概念。陰陽曆一般以陰曆的 朔望月紀月，但是12個陰曆月的日子只有354天，明顯比一個陽曆年少了11天，為了能讓陰曆的天數與真實的一年接近，無法再使用閏年的方法，因為陰曆一個月的天數是根據月亮的朔望月週期而定。

　　解決的方法，早在我國春秋時代，就已經發現235個朔望月會與19個陽曆年的天數相同，亦即在19個陽曆年之中，約每兩到三年置入一個閏月，多了閏月的那一年就會有十三個朔望月，這就是所謂十九年置七閏。西方的默冬到則要到公元前433年才發現比我國足足晚了一百六十餘年。

　　至今在世界上仍在使用的陰陽曆已經不多，其中之一即是中國的傳統曆法 - 農曆，在每個朝代的名稱不近相同，其曆法規則也有所差異，而日本民間的傳統曆法天保曆也是陰陽曆之一，實際上即是中國五代使用的元嘉曆所演化的，在西方的猶太曆或稱希伯來曆 ( Hebrew calendar ) 也是典型的陰陽曆，但其置閏規則與農曆有所不同，2009年為猶太曆的5769年。

六、 中國傳統曆法 ( Chinese Calendar )

　　中國的曆法數十部，不同的時代與不同民族都有各自的曆法，但是基本上大同小異，而今日的農曆在形式上原型為夏曆，夏曆是夏朝民間通行的曆法，可以算是我國古代最早的一部曆法，但是在夏朝之後，商朝的殷曆與西周的周曆甚至到了秦代都有改正朔的傳統，以致今日的農曆已與古老的夏曆有一定的區別。

　　中國的傳統曆法是一部陰陽合曆，須同時考慮日月運行得以紀日、紀月與紀年，且在各個朝代因民族與文化而有所差異，而曆法中主要有『歲』、『年』、『正』、『朔』四種概念。

　　中國傳統曆法的『歲』類似今日回歸年的概念，一『歲』即相當於一個回歸年，稱之為『歲實』，一『歲』之始則稱作『歲首』，『歲』的『歲首』嚴謹來說是指『天文歲首』或稱『曆算歲首』，古代將『冬至』定為『歲首』，故『歲實』即為兩次冬至之間的時間，一『歲』又分為十二個月，以『歲首』所在的月份定為『子月』，依序以十二地支紀月。

　　中國傳統曆法的『年』是指『陰曆年』，分為十二個月，每個月各以數字依序，切勿與『陽曆的年』搞混，一『年』之始也稱作『歲首』，但嚴謹來說是指『政治歲首』或稱『民用歲首』，即為今日的『春節』或『大年初一』，古代修正歲首全都是指『政治歲首』

　　中國傳統曆法的『正』為一『年』之始，即『正月』，因古時修正前朝的曆法，重定『歲首』，而名『正月』，有『修正』之意。

　　中國傳統曆法的『朔』為一『月』之始，即『朔日』，在漢朝以後，正月一日即稱之為『元朔』、『朔旦』或稱『元旦』，此指不是今日陽曆的『元旦』，切勿搞混。

　　歷史上，從先秦至漢代都有改正朔的習慣。

  • 『夏曆』的『正月』為『寅月』，即所謂的『建寅孟春之月』，『歲首』則為『子月』。
  • 『殷曆』的『正月』為『丑月』，即所謂的『建丑季冬之月』，『歲首』則為『子月』。
  • 『周曆』的『正月』為『子月』，即所謂的『建子仲冬之月』，『歲首』則為『子月』。
  • 『秦代』的『正月』為『寅月』，即所謂的『建亥孟冬之月』，『歲首』則為『亥月』。
  • 『太初曆』將『歲首』改回為『寅月』，沿用至清末。

　　西漢，落下閎、唐都、鄧平、司馬遷等人於漢武帝太初元年 ( 公元前104年 ) 制定『太初曆』。

　　民國時，孫中山先生同樣依傳統改正朔，但最大的一項改變是將國曆改為西方曆法的『格里曆』，一般稱作『新曆』，但因民間仍習慣『舊曆』，故孫中山先生以行夏正，順農時的名義，使民國同時並存兩種曆法，所以今日的農曆是承襲以往的夏正制度，正月為建寅之月。不久元旦改名為春節，而陽曆的一月一日則變成元旦，『舊曆』即被稱作『農曆』，至此可知農曆實際上並非俗稱的陰曆，也不為夏曆，就形式上被歸類為陰陽曆。

　　至今，『歲首』一詞在曆法上已不常使用，在史書上記載的『歲首』則多為『政治歲首』或『民用歲首』，但在民國時也發生分歧，『政治歲首』實質上變成陽曆的元旦，『民用歲首』則是春節。
